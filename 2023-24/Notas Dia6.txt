En el main del program(CLausuras) para poder utilizar n, debe ser estático
ya que debe existir y tener un valor los parámetros de EsMenor.
Las funciones no deben de estar rotas, por lo tanto, los returns de las funciones
deben de existir en memoria siempre, si no lo devuelve, se pierde
Ej:
int contador
int valor
return contador 
-> Contador no se pierde y se mantiene en memoria.
-> Valor se pierde ya que no se devuelve
